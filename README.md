# SNAP QC Error Viewer

🔗 **Live app:** https://zwguo.shinyapps.io/snap-qc/

## Overview
The **SNAP QC Error Viewer** is an interactive Shiny dashboard designed to explore and analyze
Supplemental Nutrition Assistance Program (SNAP) Quality Control data.

The application is built using **nationally representative SNAP QC samples**
reviewed through **state and federal audit processes**. It is designed to help users
understand how errors occur, which populations and case types are most affected,
and the financial implications of different categories of errors.

This tool is intended for researchers, policymakers, program administrators,
and analysts interested in SNAP operations, administrative burden, and program accuracy.

---

## Application Tabs

- **Error Pathways**  
  Examines how errors occurred and the underlying factors contributing to them,
  including responsibility, error type, nature, element, timing, and discovery.

- **Error Demographics**  
  Highlights which populations and case characteristics were most affected by QC errors,
  enabling analysis across demographic and case-level dimensions.

- **Error Severity**  
  Assesses the financial impact associated with different categories of QC errors,
  including overpayments, underpayments, and error magnitude.

- **Pivot Table**  
  Provides a flexible, customizable summary table that allows users to dynamically
  explore SNAP QC errors across selected dimensions, similar to a pivot-table workflow.
