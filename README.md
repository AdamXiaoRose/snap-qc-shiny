# SNAP QC Error Viewer

🔗 **Live app:** https://o.shinyapps.io/snap-qc/

## Overview
The **SNAP QC Error Viewer** is an interactive Shiny dashboard designed to explore and analyze
Supplemental Nutrition Assistance Program (SNAP) Quality Control (QC) error data.

The app allows users to:
- Examine patterns in QC errors across **states, years, and case types**
- Break down errors by **responsibility, type, nature, element, and timing**
- Interactively filter, summarize, and visualize error findings
- Use a **pivot-table–style interface** for flexible exploratory analysis

This tool is intended for researchers, policymakers, program administrators, and analysts
interested in understanding administrative errors and operational patterns in SNAP.
